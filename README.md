# andymendez.dev

Personal portfolio site — built with React 19, Tailwind CSS 4, Vite 6, and Framer Motion.

**Live:** [andymendez.dev](https://andymendez.dev)

## Stack

- **React 19** + TypeScript
- **Tailwind CSS 4** with custom dark theme
- **Vite 6** for builds
- **Framer Motion** for scroll animations
- **Shiki** for syntax-highlighted code panel
- **AWS S3** + **Cloudflare** for hosting

## Sections

- **Hero** — full-screen intro with rotating taglines
- **About** — bio, headshot, social links
- **Experience** — career timeline
- **Skills** — categorized tech grid
- **Projects** — featured personal projects with GIF demos + professional highlights
- **Contact** — contact form (serverless)

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
